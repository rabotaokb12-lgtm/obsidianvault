Prometheus
- Это инструмент который собирает метрики: CPU, RAM, время отклика сервиса, кол-во запросов в минуту и т.д

Graphana 
- Позволяет из метрик Prometheus-а создавать краисивые графики и дашборды 

ELK STack (Elastic, Logstash, Kibana)
- Elasticsearch: база данных для хранения логов с возможность. быстрый поисковых запросов. 
- Logstash - диспетчер логов, который собирает их из разных мест, фильтрует и складывается в Elasticsearch 
- Kibana: удобный веб интерфейс
Схема: Logstash -> Elasticsearch -> Kibana
