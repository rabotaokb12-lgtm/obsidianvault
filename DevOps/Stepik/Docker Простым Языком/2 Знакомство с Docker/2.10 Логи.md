**`docker stats`**: базовые метрики ресурсов (CPU, RAM, сети, диска). В РЕАЛ ТАЙМЕ
`docker stats name1 name2` - можно выводить ток определённые контейнеры
### Docker logs 
Когда контейнер запускается, всё, что приложение пишет в STDOUT/STDERR, попадает во внутренний лог Docker.
`docker logs name_container` - просмотр логов контейнера
	`-f` - трансляция логов в реал тайме
	`--tail number` - показать последние n строк
	`--since timestamp` - показать логи начиная с этого времени
		прим: --since "2026-02-03 10:00:00"
		      --since "2026-02-03"
		      --since "2026-02-03T10:00:00"
		      --since 10m   # последние 10 минут
		      --since 1h    # последний час
### Просмотр логов внутри контейнера
Если приложение не пишет в STDOUT/STDERR значит оно складывает логи в файл, напр: /var/log/app.log 
`docker exec -it container_name sh -c "tail -n 200 /var/log/app.log` - последние 200 строк 
	`docker exec -it` - заход внутрь контейнера
	`sh` - шелл внутри контейнейра (может быть и `bash`)
	`-c "comand"` - передаёт в шелл указанную команду 

`docker exec -it <container> sh -c "tail -f /var/log/app.log"` - в реал тайме логи жи есть

`docker exec -it <container> sh -c "grep -E 'ERROR|WARN' /var/log/app.log | tail -n 200"` - выведет чисто ошибки
### Отправка логов изнутри контейнера в docker logs 
#### 1 способ быстрый одноразовый
`docker exec -d <container> sh -c "tail -F /var/log/app.log >&1 &"` 
`docker logs -f <container>` 
#### 2 способ надёжный постоянный
```nginx
# Пример идеи в Dockerfile/entrypoint:
# запустить приложение и параллельно переливать лог
CMD sh -c "/usr/local/bin/start-app & tail -F /var/log/app.log"
```
#### 3 способ просмотр логов из локалки 
`docker run -v /host/logs/app.log:/var/log/app.log myapp`
	Ключевой момент тут: `-v` volume
`tail -f /host/logs/app.log` 
### Docker top и Docker events
`docker top name` - покажет какие процессы сейчас работаю в контейнере (PID, COMMAND, NAMES)
`docker events` - более продвинутая команда, позволяет слушать события (запуск остановка, создание сети, монтирование томов)
