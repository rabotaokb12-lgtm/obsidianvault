`docker exec имя_контейнера команда` - выполнить команду внутри контейнера
`docker exec -it имя контейнера /bin/bash` - запускает консоль внутри контейнера (если он установлен)
`docker cp файл имя_контейнера: путь_к_контейнеру` - копирует файл внутрь контейнера
`docker cp имя_контейнера: путь_в_контейнере локальный_путь` - копировать файл из контейнера
