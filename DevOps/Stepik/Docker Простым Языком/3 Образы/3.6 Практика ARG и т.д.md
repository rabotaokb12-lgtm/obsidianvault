## 1. Python 
### app.py
``` python 
from flask import Flask
import os

app = Flask(__name__)

@app.route("/")
def index():
    return "Hello from Advanced Dockerfile!"

if __name__ == "__main__":
    port = int(os.getenv("PORT", "8080"))
    app.run(host="0.0.0.0", port=port)
```
### dockerfile
``` dockerfile
FROM python:3.9
RUN pip install Flask==3.1.3
ARG BUILD_MODE=dev
LABEL version="0.0.1" \
      maintainer="kotakbas"
RUN useradd -M -r kotakuser
USER kotakuser
VOLUME /home/myapp/data
ONBUILD COPY . /home/myapp/data
STOPSIGNAL SIGQUIT
HEALTHCHECK --interval=30s --timeout=5s --start-period=10s --retries=3 \
  CMD curl -f http://localhost:8080/ || exit 1
SHELL ["/bin/bash", "-c"]
COPY app.py /home/myapp/app.py
EXPOSE 8080
ENTRYPOINT ["python", "/home/myapp/app.py"]
```
## 2. Nginx healtcheck и bash
### dockerfile
``` dockerfile
FROM nginx:alpine
RUN apk add --no-cache bash
SHELL ["bin/bash", "-c"]
STOPSIGNAL SIGQUIT
HEALTHCHECK --interval=30s --timeout=5s --retries=3 \
  CMD curl -f http://localhost/health || exit 1
RUN touch /usr/share/nginx/html/health
COPY index.html /usr/share/nginx/html/
CMD ["nginx", "-g", "daemon off;"]
```
## 4. Python Flask + healthcheck + volume logs
Прикол этого контейнера в том что он записывает логи в папку с dockerfile в папку logs
### dockerfile
```dockerfile
FROM python:3.9
RUN useradd -M -r webuser
WORKDIR /app
COPY requirements.txt app.py .
RUN pip install -r requirements.txt
RUN mkdir -p /data/logs && chown -R webuser:webuser /data/logs
HEALTHCHECK --interval=30s --timeout=5s --retries=3 \
  CMD curl -f http://localhost:5000/ || exit 1
VOLUME /data/logs
EXPOSE 5000
CMD ["python", "app.py"]
#Можно было ещё вписать выше USER webuser НО тогда логи не записываются из за отсутсвия прав
```
**Запуск команды**
`docker run -d -p 5000:5000 --name name -v $(pwd)/logs:/data/logs imagename`

### app.py
``` python
from flask import Flask
import logging
import os

LOG_DIR = "/data/logs"
LOG_FILE = os.path.join(LOG_DIR, "app.log")

# Создаём папку если её нет
os.makedirs(LOG_DIR, exist_ok=True)

logging.basicConfig(
    filename=LOG_FILE,
    level=logging.INFO,
    format="%(asctime)s [%(levelname)s] %(message)s",
)

app = Flask(__name__)

@app.route("/")
def index():
    app.logger.info("Main page accessed")
    return "Hello from a health-checked Flask!"

@app.route("/health")
def health():
    app.logger.info("Health check called")
    return "OK", 200

if __name__ == "__main__":
    app.logger.info("Application started")
    app.run(host="0.0.0.0", port=5000)
```
