### git status
#gitstatus
`git status --porcelain | wc -l` -
	`git status --porcelain` - покажет какие файлы не закомиченны, находятся локально типо пока что 
	`ws -l` - покажет сколько есть таких файлов 
	`git status -s` - краткая форма вывода 
### git add
#gitadd
**Stagging area** - промежуточный этап до коммита, нужен чтобы коммитить только определённые файлы 
`git add filename` - добавить как раз таки в этот stagging area
	`git add .` - добавить все файлы туда 
	`git add *.log` - если нужно добавить все файлы с таким расширением
	`git add -A` - добавить все изменения в репу
	`git add -p` - интерактивное добавление частей файла 
### git diff
#gitdiff
`git diff --cached --name-only` 
	`git diff` - покажет какие файлы изменены и что было изменено но ещё не находится в *staging area* 
	`--cached` - покажет какие изменения уже есть в *staging area*
	`--name-only` - ну понятно ток имена файлов
	`git diff filename` - статус конкретного файла
	`git diff HEAD~1` - сравнение с конкретным коммитом
	`git diff --staged` - изменения в индексе
	
`git diff README.md | grep "^+" | grep -v "^+++" | wc -l` - покажет кол-во изменённых строк 
`git diff file | wc -l` - покажет сколько символов добавилось в файл 

### git commit 
#gitcommit
`git commit` - коммит с редактором сообщения
	`git commit -m "text"` - сразу текст ввести 
	`git commit -am "Text"` - коммит всех изменённых отслеживаемых файлов
	`git commit --amend` - изменение последнего коммита
`touch .gitignore` - создание файла игнорщицы и туда уже вписываем названия или расширения файлов которые будут проигнорированы, это нужно чтобы они не высвечивались при вводе команды `git status` (ну тип чтобы не показывалось какие файлы не закомиченны) 
### git log
#gitlog
`git log --oneline | head -2` 
	`git log` - показывает инфу о всех коммитах
	`--oneline` - флаг которые покажет нам только id коммита и его описание 
	`| head -2` - первые 2 строчки из этой команды

```shell
echo "Commits: $(git rev-list --count HEAD), Tracked files: $(git ls-files | wc -l), Ignored: $(git status --ignored --porcelain | grep '^!!' | wc -l)"
```
Легче показать результат чем объяснить :
Commits: 2, Tracked files: 3, Ignored: 1

`git log --oneline -2 --pretty=format:"%s"` - покажет название 2х последних коммитов
