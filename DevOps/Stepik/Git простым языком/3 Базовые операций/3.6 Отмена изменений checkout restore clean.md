## git restore
`git restore файл` - откатывает изменения в файле до последнего коммита 
	`git restore .` - все файлы откатить 
	`src/` - все файлы в папке
	`file file` - несколько файлов
`git restore --staged .` - убрать файлы из индекса добавленной командой git add
`git restore --source=HEAD~2 README.md` - убирать можно не только до последнего коммита а до любого 
`git restore --source=feature-branch config.json` -
Восстановить файл из другой ветки

## git checkout
Старая версия git restore 
Прим: 
`git checkout -- .` 
## git clean
Удаляет не отслеживаемые файлы 
`git clean -n` - посмотреть какие файлы удаляться 
	`git clean -f` - удалит не отслеживаемый файлы
	`git clean -fd` - файлы и директорий
	`git clean -fx` - удалит и игнорируемые файлы тоже
	`git clean -i` - интерактивный режим 
	Можно и без файлов `-x` `-d`
