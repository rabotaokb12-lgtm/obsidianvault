- Для Ubuntu/Debian используйте команду:  
    `sudo apt update && sudo apt install git -y`
- Для CentOS/RHEL используйте:  
    `sudo yum install git -y`
- Для Arch Linux:  
    `sudo pacman -S git`

После установки убедитесь, что Git успешно установлен, командой:  
`git --version`

Перед работой нужно настроить имя и емайл, чтобы в комитах было понятно кто их делал
```csharp
git config --global user.name "Ваше Имя"
git config --global user.email "ваш_email@example.com"
```
--global - скорее всего означает то что ВЕЗДЕ будет такой ник
Проверьте, что настройки сохранились:
```css
git config --global --list
```
### Настройка редактора по умолчанию
Git использует текстовый редактор для создания коммитов и некоторых операций. Вы можете установить удобный вам редактор, например, nano или vim:
- **nano:**  
    `git config --global core.editor "nano"`
- **vim:**  
    `git config --global core.editor "vim"`
### SSH ключи где и как создать
`~/.ssh/` - путь к ключу
1. `ssh-keygen -t ed25519 -C "email@email.com"` - генерация ключа
	`ssh-keygen -t ed25519 -C "email@email.com" -f ~/.ssh/имя ключа` - генерация с именем файла
2. Запуск SSH агента и добавление ключа 
   `eval "$(ssh-agent -s)"`
   `ssh-add ~/.ssh/имя ключа`
3. `cat ~/.ssh/имя ключа.pub` копируем и вставляем на гите (SSH ключи в настройках)
