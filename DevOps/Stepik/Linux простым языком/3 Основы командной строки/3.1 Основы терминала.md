## Справочник команд 
``man имя команды`` - мануал по команде 
	- Стрелки или PgUp/PgDn, чтобы прокручивать текст.
	- `/` — поиск по ключевым словам (после ввода `/` наберите слово и нажмите Enter).
	- `Q` — выход из просмотра.
``имя команды --help`` - показывает краткое описание и список доступных опций 
``tldr имя команды`` - альтернативный мануал с примерами т.д
	Перед использованием нужно установить
	``sudo apt install tdr``
