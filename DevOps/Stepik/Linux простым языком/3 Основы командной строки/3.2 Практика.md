## 3.21 Спец права
### SUID (Set User ID)
Если на исполняемом файле установлен SUID  то программа запуститься с правами владельца 
	Пример: passwd - владелец root но у него стоит SUID и поэтому остальные юзеры могу менять свой пароль
`chmod u + s ` `chmod 4755` - тут именно 4 за это отвечает
### SGID 
Аналогично SUID, но распространяется на _группу_
Если каталог имеет SGID, то все новые файлы и подпапки, созданные в нём, наследуют группу каталога (а не основную группу пользователя, создавшего файл)
`chmod g + s` `chmod 2755` - тут за это отвечает 2
### Sticky bit
Если на каталоге выставлен *Sticky bit* то редактировать файлы внутри каталога сможет ТОЛЬКО ВЛАДЕЛЬЦЫ ЭТИХ ФАЙЛОВ 
	В выводе `ls -l` вместо `x` у `others` будет буква `t`. Например, `drwxrwxrwt`.
`chmod +t ` `chmod 1755` - тут за это отвечает 1

## 3.27 Архивация в Linux - tar 
### Базовая логика
**Без сжатия**: Команда `tar` умеет складывать много файлов в один `.tar` 
**Со сжатием**: При желаний можно тут же сжать архив с помощью *gzip*, *bzip2*, *xz*
- `.tar` — просто пакет файлов без сжатия
- `.tar.gz` или `.tgz` — tar-архив, сжатый gzip
- `.tar.bz2` — tar, сжатый bzip2
- `.tar.xz` — tar, сжатый xz
### Команды архива 
`tar` 
	`-c`: create
	`-z`: gzip сжатие
	`-v`: подробный вывод что пакуем
	`-f`: указывает что далее идёт имя файла архива
	`-J`: xz сжатие 
	`-j`: bzip2 сжатие
	`tar -czvf archive.tar.gz folder/` 
	`-x`: x-extract распаковка
	`tar -xzvf acrcive.tar.gz`  
	`-t`: показывает что внутри архива не извлекая файлы
	`tar -tzvf acrchive.tar.gz` 
### Распаковка архива 
`tar -x`: x-extract 
	`tar -xzvf acrcive.tar.gz` 
	`-C` - указать путь куда распаковать
	`tar -xzvf archive.tar.gz -C путь`
	`-p` - сохранить все права за владельцем
## 3.28 Сжатие данных
Все эти команды сжимают лишь один файл ну кроме **zip**
### gzip/gunzip
`gzip file` - fle.gz сжатие
`gunzip file.gz` - распаковка 
**По умолчанию** удаляет оригинальный файл
`-k` - сохранить оригинал
`gzip -1` (или `--fast`) — быстро, но с меньшей степенью сжатия.
`gzip -9` (или `--best`) — максимальное сжатие, но медленне
### bzip2/bunzip2
`bzip file` - file.bz2
`bunzip file.bz2` - распаковка
`bzip2 -t file.bz2` - проверяет целостность сжатого файла
