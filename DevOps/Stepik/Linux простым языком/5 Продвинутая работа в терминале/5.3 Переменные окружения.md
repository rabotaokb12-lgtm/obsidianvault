`printenv` - показывает все переменные окружения 
`MY_VARIABLE="Helo world"` - создание переменной
	Параметры могут быть разными
	Прим:
	USER=alice
	HOME=/home/alice
	PATH=/usr/local/sbin:/usr/local/bin:/usr/sbin:/usr/bin
	SHELL=/bin/bash
`export MY_VARIABLE` - теперь она доступна всем программам
`export MY_VARIABLE="text"` - в одной комманде 
`cat $MY_VARIABLE` - пример использование
`unset MY_VARIABLE` - удаление 
#### Самые важные переменные 
**PATH** - список директорий, где система ищет исполняемые файлы (команды и приложения):
	`echo $PATH`
**USER** - имя текущего узера
	`echo $USER`
**HOME** - домашний каталог пользователя
**SHELL** - какой шел юзается 
**EDITOR** - редактор текста по умолчанию
	`expoty EDITOR=nano`
**LANG** - языковые настройки окружения
#### Постоянная переменная
`nano ~/.bashrc` 
`export MY_VARIABLE="txt"` - в конец файла
Сохранить файл
или 
Откройте файл настроек:

```bash
vim ~/.bashrc
```
В конец файла добавьте строку с вашей переменной:

```bash
export MY_VARIABLE="Моё постоянное значение"
```
Сохраните файл и примените изменения командой:

```bash
source ~/.bashrc
```
