`grep "name" file` - поиск слова в строке, покажет строки с этим текстом #grep 
`-i` - игнорировать регистр букв
`-r` - искать это слова во всех файлах каталога
`-n` - показывать номер строки
`-l` - показать только имена файлов где был найден текст
`-H` - показывает имя файла перед строкой с совпадением
`-v` - находит строки в которых НЕТ совпадения
`-w` - покажет только слова совпадения 
`--color=auto` - подсветит совпадение

Пример комбинирования команды grep 
`ps aux | grep nginx`

### Регулярные выражение 
`grep "^Linux" notes.txt` - покажет все строки начинающиеся со слова Linux
- `^` — начало строки
- `$` — конец строки 
	`grep "completed$" tasks.log`

- `.` — любой одиночный символ (кроме переноса строки)
	- `grep "c.t" words.txt`

- `*` — повторение предыдущего символа 0 или больше раз `grep "doc.*txt" files.log`

- `[]` — диапазон символов (например, `[a-z]` или `[0-9]`) `grep "file[0-9].txt" list.txt`

- `\` — экранирование специального символа
  `grep "version\.txt" files.txt`


- `|` — или (логическое OR)
  `grep -E "error|failed|denied" syslog`
  	Access denied
	Operation failed
	Network error

- `()` — группировка
  `grep -E "(ha)+" laughter.txt`
	haha
	hahaha
	hahahahaha
- `+` — одно или более повторений

- `?` — ноль или одно повторение (необязательно)
