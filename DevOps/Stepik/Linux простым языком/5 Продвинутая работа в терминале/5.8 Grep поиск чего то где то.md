`grep "name" file` - поиск слова в строке, покажет строки с этим текстом #grep 
`-i` - игнорировать регистр букв
`-r` - искать это слова во всех файлах каталога
`-n` - показывать номер строки
`-l` - показать только имена файлов где был найден текст
`-H` - показывает имя файла перед строкой с совпадением
`-v` - находит строки в которых НЕТ совпадения
`-w` - покажет только слова совпадения 
`--color=auto` - подсветит совпадение

Пример комбинирования команды grep 
`ps aux | grep nginx`
