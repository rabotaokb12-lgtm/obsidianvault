### 1 Структура 
``` bash
#!/bin/bash
echo "Привет мир"
код...
```

- `#!/bin/bash` — это **shebang** (шаблон запуска), указывающий системе, что скрипт следует выполнять в оболочке Bash.

После `#!` **сразу** идёт путь к интерпретатору без пробела: `#!/bin/bash` или `#!/usr/bin/env bash`.
### 2 Как запускать скрипт 
```bash
./script.sh          # запуск как программы (нужен chmod +x)
bash script.sh       # явный вызов интерпретатора
source script.sh     # выполнить в текущей оболочке (не создаёт новый процесс)
```
### 3 Переменные в bash
1. Имя переменных заглавными
2. Для присвоения `NAME=name`(слитно всё)
3. Обращение идёт через `$` 
4. Подставляя переменную в путь пишутся "" `echo "$FILE"`
Пример использования переменных:
```bash
#!/bin/bash

NAME="Антон"
AGE=30

echo "Меня зовут $NAME, мне $AGE лет."
```
### 4 Передача аргументов в скрипт 
**Bash** может принимать аргументы из командной строки 
- `$1, $2 и т.д` - первый второй и послед. аргументы
- `$0` - имя скрипта
- `$#` - кол-во переданных аргументов
Пример скрипта с аргументами:
```bash
#!/bin/bash

echo "Имя скрипта: $0"
echo "Первый аргумент: $1"
echo "Количество аргументов: $#"
```

`./args.sh Linux Bash` - запуск скрпита
