`CTRL + ALT + F2` - переход в терминальный режим при загрузке системы

Если есть ещё чутка памяти 
`sudo apt install cloud-guest-utils`
`sudo growpart /dev/sda 3`
`sudo resize2fs /dev/sda3`

При отсутствий памяти:
`sudo apt-get autoremove --purge -y`
`sudo apt-get clean`
`sudo journalctl --vacuum-size=100M`
`sudo rm -rf /var/tmp/*`
`sudo apt-get remove имя пакета`

Проверка памяти 
`df -h` - покажет скок осталось памяти
`sudo du -h /var/log --max-depth=1 | sort -hr` - покажет сколько весят файлы в текущей директорий 
`du -h --max-depth=1 2>/dev/null | sort -hr | head -10` - топ 10 самых тяжелых директорий
