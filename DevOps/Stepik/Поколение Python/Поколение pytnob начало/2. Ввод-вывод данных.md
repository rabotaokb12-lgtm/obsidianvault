### 2.2 2.3 Команда print-печать input-ввод данных
**print(" ")** print(' ') print(" ", переменная )
переменная = **input()**
Для ввода числовых типов данных: переменная = **int(input())**
**sep("")** - замена пробелов в print()
**end("")** - замена новой строки в print()
Пример: 
	a = " питух"
	print("Пошёл нахуй", a, sep = "," , end = "." )
	print(" Понял?")
	Вывод: Пошёл нахуй, питух. Понял?

### 2.4 2.5 Целочисленная арифметика
**ТАБЛИЦА ПРИОРИТЕТОВ АРИФМЕТИКИ В ПИТОН**

| ()               | ну что сказать, скобки                                        |
| ---------------- | ------------------------------------------------------------- |
| **               | возведение в степень                                          |
| -(унарный минус) |                                                               |
| // / % *         | целочисленное деление, деление, остаток от деления, умножения |
| + -              | плюс минус                                                    |
|                  |                                                               |
Примеры
	25 // 12 = 2
	-25 // 12 = -3 (да не удивляйся, ебанутая хуйня, округление в меньшую сторону типа)
	25 // 26 = 0
	25 % 12 = 1
	25 % 26 = 25 
	-25*.* 2 = -625 (да один хуй будет минус) а если
	(-25)* * 2 = 625 
**Алгоритм для нахождения каждой цифры в n - значном числе**
	- Последняя цифра: `num % 10`
	- Предпоследняя цифра: `(num // 10) % 10`
	- Предпредпоследняя цифра: `(num // 100) % 10`
	- ...
	- Третья цифра: `(num // 10 ** (n - 3)) % 10`
	- Вторая цифра: `(num // 10 ** (n - 2)) % 10`
	- Первая цифра: `num // 10 ** (n - 1)`
	Пример c цифрой 123:
		123 % 10 = 3
		(123 // 10) % 10 = 12 % 10 = 2
		(123 // 100) % 10 = 1 % 10 = 1
		...
		123 // 10 ** 0  % 10= 123 // 1 % 10 = 123 % 10 = 3
		123 // 10 ** 1 % 10 = 123 // 10 % 10 = 12 % 10 = 2
		123 // 10 ** 2 = 123 // 100 = 1
