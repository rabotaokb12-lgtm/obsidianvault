### Установка из неофициального репозитория
sudo add-apt-repository **ppa**:deadsnakes/ppa   
**ppa** - personal packages application

### Установка nodejs(чтобы была более новая версия)
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
`curl` — утилита для скачивания данных по ссылке.
| - труба что говорит bash скрипту бери текст из curl
sudo : запуск скрипта с правами админа
-E : при переходе в root сбрасываются переменные окружения, типо $PATH $LANG $HOME а эта команда говорит не сбрасывать переменные окружения(карочь чтобы скрипт знал нужный путь и т.д)
bash - : выполнение shell-скрипта 

В общем команда говорит "Скачай новую версию из этого сайта используя права супер пользователя, не убирая переменные окружения, выполни через bash скрипт, взяв текст для скрипта из начальной команды"

Затем только пишешь 
sudo apt install nodejs

Всё это было сделано потому что в дебиан репозиторий обычно лежит старая версия nodejs а тут она новая
