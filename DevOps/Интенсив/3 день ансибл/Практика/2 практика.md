### 1. В файле `playbooks/deploy.yml`:
#### 1.1 Добавить бэкап текущих конфигураций (модуль copy с remote_src)
``` yaml
    - name: Бэкап deploy.yaml
      copy:
        src: "{{ config_dir }}/"
        dest: "{{ backup_dir }}/config_dir.backup/"
        remote_src: yes
        directoty_mode: '0755'
```
Объяснение:
	`copy` - модуль нужный для копирования чего либо 
		просто выставляй src и dest
	`remote_src` 
		yes: файл лежит на сервере
		no(по дефолту): файла лежит на моей машине
#### 1.3 Добавить генерацию database.yml при db_type == 'postgresql'
``` yaml
    - name: Генераций БД из шаблона
      template:
        src: "../templates/backend/database.yml.j2"
        dest: "{{ config_dir }}/database.conf"
      when: db_type == 'postgresql'
```
