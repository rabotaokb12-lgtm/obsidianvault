## Ошибка пароля PSQL #psql
При выходе ошибки постгресс ноу пассворд
cd /etc/postgresql/< version>/main
nano pg_hba.conf
Найти строки для `local` и заменить `md5` на `trust`, например:
local all postgres trust

## Ошибка Nginx 
nginx: [emerg] a duplicate default server for 0.0.0.0:80 in /etc/nginx/sites-enabled/default:22
Значит что 2 сервака пытаются слушать 80 порт одновременно решение 
1. Чекнуть сперва 
	   `cd /etc/nginx/sites-enabled`
	   `nano default`
2. Если так оно и есть вводим s
	   `sudo rm /etc/nginx/sites-enabled/default`
	   `sudo nginx -t`
	   `sudo systemctl reload nginx`
