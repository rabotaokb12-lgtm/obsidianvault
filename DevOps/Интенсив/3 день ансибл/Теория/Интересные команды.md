Как запушить команду если работа была в линукс 
cd /путь/к/твоему/проекту
git status
git add название файла или чтобы все изменения git add .
git commit -m "Заметка"
git push

При выходе ошибка 
git pull --rebase origin main
git push

Как запушить изменения из веб морды в линукс 
Переходим в папку и прописываем
git pull

## Проверка подключения инвентори файла ансибл 
``ansible -i inventory/localhost.ini all -m ping`` 
## Команда для проверки синтаксиса ямл файлов ансибл
`ansible-playbook -i inventory/localhost.ini playbooks/site.yml --syntax-check`
